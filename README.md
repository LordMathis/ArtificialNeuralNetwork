# Artificial Neural Network

Java implementation of multilayer perceptron

## Features

* Mini-batch gradient descent
* Quadratic cost
* Cross-entropy cost
* Sigmoid activation
* L2 regularization
* Momentum

## Usage

Requires maven

```
mvn clean compile
mvn exec:java -Dexec.mainClass=com.namesny.Main
```

See examples for more
